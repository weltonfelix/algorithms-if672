# A Maior das Listas

Dadas `N` listas com elementos separados por `,`, imprima a maior delas.

## Input Specification

Seguem `N` linhas, cada uma contendo uma lista de elementos separados por `,` e
delimitada por `[` e `]`.

## Output Specification

Imprima a maior das `N` listas, como dada na entrada.

## Notes

Em caso de empate, deve-se imprimir a primeira das `N` listas que possui tal
maior nÃºmero de elementos.

> [!IMPORTANT]
> Time Limit: 1 sec  
> Memory Limit: 768 MB  
> Output Limit: 4 MB

## Examples

### Sample #1

#### Input

```text
[1,2,3]
[5,6]
```

#### Output

```text
[1,2,3]
```

### Sample #2

#### Input

```text
[True,False,False]
[True,True,True,True]
[False]
[True]
```

#### Output

```text
[True,True,True,True]
```
